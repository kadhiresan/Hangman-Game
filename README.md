# A Hangman-Game with NodeJs(Sails) and AngularJs

a [Sails](http://sailsjs.org) application

## Installation

1. Clone the repo

  ```bash
    git clone https://github.com/kadhiresan/Hangman-Game.git
  ```

2. Build installation
  ```bash
  npm install
  ```

3. Run the project
  ```bash
  sails lift
  ```
